# 量子航法で超高精度な位置測定　〜冷たい原子がGPSを置き換える？〜

GPSが使えない海の中や地下でも、超高精度に自分の位置がわかる——そんな夢のような技術「量子航法」が現実になりつつあります。

この動画では、原子を極低温まで冷やして「波」として扱い、その干渉パターンから加速度を測る「原子干渉計」の仕組みを基礎から解説。高校物理で習う積分や波の干渉といった概念が、どのように最先端技術につながるのかを紹介します。

🔍 この動画でわかること：
・ 慣性航法とは？ 加速度を2回積分して位置を求める原理
・ なぜ加速度の誤差が「時間の2乗」で位置誤差に蓄積してしまうのか
・ GPSの弱点：水中・地下では使えない、スプーフィングの脅威
・ 冷却原子とド・ブロイ波——物質の「波としての性質」
・ レーザー冷却で原子を絶対零度近くまで冷やす方法
・ 原子干渉計の仕組み
・ 光学干渉計（マッハ・ツェンダー）との対応関係
・ 従来のMEMS加速度計と比べて4桁以上の精度向上
・ 2025年の実用化最前線
