# 量子航法で超高精度な位置測定　〜冷えた原子がGPSを置き換える？〜

GPSが使えない環境でも、超高精度な位置測定を可能にする「量子航法」。冷却原子と原子干渉計を使った次世代の慣性航法技術について、基礎から最新動向まで解説します。

🔍 この動画でわかること：
・ 慣性航法の基本原理と誤差蓄積問題
・ なぜ冷却原子を使うのか（コヒーレンスの重要性）
・ 原子干渉計の仕組み（ラマン遷移とπ/2-π-π/2パルス）
・ 量子センサーがMEMSに勝る理由
・ 2025年の実用化動向（Boeing、Q-CTRL等）
